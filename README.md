# Single-Page Web Demos

A curated collection of **single-page websites** built to showcase **front-end development skills**. Each project focuses on clean layout, responsive design and modern UI practices.

# Repository Structure

```
/project-name-1
    ├─ index.html
    ├─ style.css
    └─ README.md
/project-name-2
    ├─ index.html
    ├─ style.css
    └─ README.md
...
README.md
```

* Each folder contains a **single-page project** with its own HTML, CSS and optional JS files.
* Projects are self-contained, making it easy to view, edit or reuse components.

# Features Highlighted

* Responsive design for desktop and mobile
* Modular CSS structure
* Modern layout and typography techniques
* Clean and maintainable code

# How to Use

1. Clone the repository:

```bash
git clone https://github.com/rocksoncodes/Single-Page-Web-Demos.git
```

2. Open any project folder and launch `index.html` in your browser.
3. Explore the code to understand the structure and styling techniques.
